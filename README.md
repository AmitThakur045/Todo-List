# ToDo-List

ToDo list is a simple web application to save Your daily tasks in order not to miss anything.

### Features
* Add new task
* Mark task as completed / uncompleted
* Edit task
* Delete single task
* Delete completed tasks
* Delete all tasks
* Add different todo-list

### Technologies used:
* HTML
* CSS
* JavaScript
* ejs
* NodeJS
* MongoDb
* Mongoose
